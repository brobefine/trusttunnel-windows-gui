Place required app icons here:
- StoreLogo.png      (50x50)
- Square150x150Logo.png
- Square44x44Logo.png
- Wide310x150Logo.png
- LockScreenLogo.png
- SplashScreen.png   (620x300)

For an unpackaged app (WindowsPackageType=None) only StoreLogo.png is strictly used,
but having the others avoids MSBuild warnings. Generate quickly with any image tool.
